# Mario's Pizza & Registration

---

## Description

Created an **Mario's Pizza & Registration** using React JS
This project includes two main parts a User Registration Form and a Pizza Ordering System.
In the registration form, users can enter their basic details.  
In the pizza ordering section, users can select pizza size, crust type, toppings, and quantity.
The total price updates automatically based on the selected options, and an order summary is displayed.
